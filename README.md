# 1Office AI-First Landing Page

File chính để GitHub Pages đọc là `index.html`.

## Cách sửa nhanh trên GitHub
1. Mở repository trên GitHub.
2. Click file `index.html`.
3. Bấm biểu tượng cây bút / Edit this file.
4. Tìm đoạn text cần sửa bằng Cmd+F hoặc Ctrl+F.
5. Sửa nội dung.
6. Bấm Commit changes để lưu.
7. Chờ 1-2 phút rồi refresh link GitHub Pages.

## Lưu ý
- Không đổi tên `index.html`.
- Mỗi lần sửa nên commit message rõ ràng, ví dụ: `Sửa headline section 3`.
- Nếu sửa hỏng, có thể vào History để xem lại các phiên bản trước.
